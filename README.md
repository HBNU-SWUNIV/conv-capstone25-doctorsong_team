## <u>Teamate</u> Project Background
- ### 필요성
  - 공원 내 쓰레기 투기 및 흡연 문제는 시민 안전과 환경 위생을 저해하며, 관리 인력과 비용이 꾸준히 증가하고 있다.
  - 자율주행 기반 로봇 기술은 발전했지만, 실제 쓰레기 수거를 수행할 수 있는 로봇팔 기반 시스템은 부재하다.
- ### 기존 해결책의 문제점
  - 기존 공원 로봇은 단순 이동, 배달 등 제한된 역할에 머무르며, 쓰레기 수거 기능을 수행하지 못한다.
  - 단순 청소 로봇은 주행 중심으로, 실질적인 ‘집기 및 분리수거’ 작업을 자동화하지 못하고 여전히 인력 의존도가 높다.
  
## System Design
  - ### System Requirements
    - ROS 기반의 로봇팔 제어 및 MoveIt을 이용한 경로 계획과 역기구학 연산 기능 구현
    - 인식 좌표와 로봇팔 작업좌표의 정밀한 TF 변환 및 FSM 기반 자동화 동작(집기–운반–적재)
    
## Case Study
  - ### Description
  - OpenManipulator-X 로봇팔과 Realsense Depth Camera, LiDAR를 결합한 쓰레기 인식 및 집기 시스템 구축
  - Gazebo 시뮬레이션을 통해 로봇팔 제어 알고리즘을 검증하고, 실제 공원 환경과 유사한 조건에서 프로토타입 실증 수행
  - 다양한 쓰레기 크기/형태/위치에서의 그립 성공률 및 안정성 평
  
## Conclusion
  - 비정형 실외환경에서도 쓰레기 탐지, 집기, 분리수거가 가능한 지능형 로봇팔 기술을 확보하였다.
  - 공공관리 자동화 및 스마트시티 구축의 기반 기술로 발전 가능하며, 향후 캠퍼스·공항·상업시설 등으로 응용 확장될 수 있다.
